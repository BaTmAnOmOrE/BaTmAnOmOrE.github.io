    Learn about the diffences between the padding, margin and border

    introduction to Padding Margins and Borders


    When adding styles to an HTML element with CSS, it is treated like a box by defaults. The Border, margin, and padding are the properties that affect appearance of the box’s edges. Simply the margin is the area on the outside of a box’s border and padding is the area on the inside of a border. The main purpose is to add space between elements for cleaner design and viewing.


    The padding has 4 differnt properties; The top,The Left, The Right and The Bottom

    For example if you were to adjust all sides of the padding for all paragraphs in this blog post You would add this to your CSS file (Cascading Style Sheet aka How you make a website pretty).


    p
    {
    padding: 20px;
    }


    Using the above code would allow you to change the padding for the Top,Left,Right and Bottom by 20 pixels







    The margin properties are used to create space around elements


    Just like the padding, the margin has 4 diferent properties the Top,Left, Right and Bottom


    p
    {
    margin-top: 90px;
    margin-bottom: 90px;
    margin-right: 100px;
    margin-left: 75px;
    }

    The above code affects the margin properties.
    for example the margin-top: 90 px will add 90 pixels of space to the top margin property

    you get the jist
